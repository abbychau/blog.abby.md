就我的圈子而言的普偏偏見 2

2013-10-08
日常生活

1. 總以為香港的新一代很沒救, 很不能自立, 對社會很沒認識. 除了蘋果日報的報道真不知道這從何而來
2. 香港樂壇已死 < 發展到盡頭就是如此, 一日還有出產廣東歌就好, 音樂環境太差, 但死亡兩個字就是偏見的源頭和偏見成果
3. 大陸的軟件有木馬 < 用產地來分真是為了節省腦部動能吧
4. 山寨手機品質不好 < 山寨與否還不是用那無腦的人的 淺薄認知來戒定
5. PHP 開發時必要有模版引擎 < PHP 本來就是模版語言, 形式化到這個地方就是MK
6. i3 是處理效能低下 < 這是因為你們都對3 字比起5字和7字都幾乎差了一倍的這感覺
7. Smartphone適合玩遊戲 < 沒有硬鍵反饋跟本玩不到真正意義上的遊戲
8. Windows 是不穩定的作業系統 < Zune 是失敗的產品, Surface 沒用, MSSql 慢, VB 是玩具, 只要黑微軟就政治正確了?
9. 微軟是邪惡的公司 < 同上
10. Mac OS X十分優雅 < Mac OS 有點陣形文字嗎? 如此花費GPU 如此注重那麼搶眼的統一風格, 這是OS 越權了
11. Desktop/Notebook 已經沒落 < Netbook 會, tablet 也會, 但多工和鍵盤的無Delay 的可靠性才是necessity, 其他只是令消費者舒服的Luxury 而已
12. 中學以上的數理化三範很難, 在日常生活也不會用到 < 不會才不用, 但不表示你用不到; 社會容許你不應用地生活, 但也給你應用後的便利
