四川麻將定缺為 1+1+2 的時候是 1 容易贏還是 2 容易贏？

2015-02-15 14:50
答案
影響牌種是清一色/清對/清七對/青龍七對 中間會搶牌
設以該牌種的機率為P(c),
中間以筒的機率為P(c|t) , 索為P(c|s), 萬為P(c|m)
又稱為 p(t)/p(s)/p(m)

A: 0.5p(t) + 0.5p(s)
B: 0.5p(s) + 0.5p(m)
C: 0.5p(t) + 0.5p(m)
D: 0.5p(t) + 0.5p(m)

A為p(s) 時 => 威脅為0.5B
A為p(t) 時 => 威脅為0.5C+0.5D

B為p(s) 時 => 威脅為0.5A
B為p(m) 時 => 威脅為0.5C+0.5D

C為p(t) 時 => 威脅為0.5A+0.5D
C為p(m) 時 => 威脅為0.5B+0.5D

D為p(t) 時 => 威脅為0.5A+0.5C
D為p(m) 時 => 威脅為0.5B+0.5C

A 在做索子時 或 B做索子時 會只有 50%威脅度
A/B 兩家會在牌面半均時, 選擇p(s), 因為最差的結果也只是
A: 1B / B: 1A
所以, A/B 和的牌面較大的的幾率比較高
總括長時間當A/B 家, 贏的機率也較高
