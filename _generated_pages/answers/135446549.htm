為什麽用php的人越來越少了？

2016-12-12 04:16
答案
最近一年，可以看出C 和C++ 互換一些項目了。餘下的一些可能被swift 吃了。

Python 是受益於大數據、ML和DL 的xNN庫的冒起穩步上揚。

其他的語言都把佔有率奉獻給Java 了。
得益於Clojure、Scala 以至Kotlin 的成功，大量的輪子出現了。而日益普及的第三世界的安卓手機市場和安卓的高速叠代也為Java 提供了市場能源。

集中看php ，由於laravel 的出現，和包管理的普及，令php 的輪子處於很穩定的狀態，底層的改動已經沒有往日頻繁。但最大問題還是在於伺服器端的重要性被BaaS 、功能越來越強大的資料庫（例如Meteor)、和Client Server 削弱了不少。而偏偏99%的php 應用都是因伺服器而生。而且新應用也對nodejs 十分友好，PHP 的重要性和市場大小也就受到限制。

但如果你需要一個語言功能完整、開發迅速、而且發佈簡便的伺服器端語言，PHP 還是一個十分好的選擇。
