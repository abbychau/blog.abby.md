為什麽學 C++ 的看不起學 Java 的？

2015-12-04 17:49
答案
豈只c++看不起java 。

java 本來是種又易學，又簡單，又統一的語言。
可是sun 和oracle 這兩家公司確實不爭氣。
他們和商業靠得太近了，導致java 世界出現了很多業務導向的機能。
hibernate 加structs 兩大巨型框架簡直是把oo ，不，是把ide 都濫用了。

如果php 是初期預期偏差而成長而成的四不像怪物。
java 就像是個人才卻被各路鬼畜調教而成的怪物。

現今常用的不論任何語言，都從未有過像java 那麼沈重的商業任務。(cobol 我當它死了)
這導致java 新特性的成長一點也不cool
