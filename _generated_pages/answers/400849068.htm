知乎上這麽多推崇學 Python 入 IT 行的，如果他們學完 Python 這一套找不到工作怎麽辦？

2018-05-25 06:00
答案
因為很多人初學者都會問: 我想學編程，學甚麼語言好？

那總究要有個答案呀，去讓初學者先跑個hello world。

那提議哪個好呢？
C 是挺好的，但指針繞不過去有錯敗感啊，還有一堆越界和返回記憶體位置，都是錯敗感。
C++ 和Rust ，這用來嚇跑初學者還差不多。
PHP，好是挺好但不是每個人都想做網頁。
C#/Java，好是挺好但環境配起來要給他們補課啊。
Scala / Elixir: 漂亮是漂亮但造出來的東西不夠滿足感。

Python ，這東西慢歸慢，要拿它認真造個遊戲註定要死，但到那個時候都入門了。而且過程中形態不太強，數組夠簡便，這就OK 了。

就算說上面的任何一個語言，也不是說你學了就可以找到工作的。
學了遊泳也應徵不了救生員。就算你遊得夠快了，樣子嚇壞人也是不行的啊。

學東西和找工作是兩回事啊。
