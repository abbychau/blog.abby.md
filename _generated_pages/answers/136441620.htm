如何評價 NVIDIA 發布的 DGX-1？

2016-12-18 08:54
答案

已經是大半年前的新聞了, 但感覺留意的人不多。



這是



雖說Tesla P100 已經是為伺服器使用Blower 設計, 但效率上還是不及中央風扇好, 而且也浪費空間。


DGX 是採用像blade server 一樣的純中央風扇

幾乎所以機櫃面的操作空間都留給了SSD 的儲存和更換



3200W x2

官網上放過的這張


所謂為HPC 而設的P100, 其實根本除了Nvidia 自己根本沒有人會用得上

現在DGX-1 就像是為它做了個示範

然後就是這臺機的大殺器, 脫離PCI-E 之後, 可以愉快地使用自家的NVLink 來做CPU<>GPU 的80xPCIE速度交換

為clustering 做好準備的 4x 100Gbps InfiniBand

E5-2698 v4, 要註意NV 的比對的75倍DL 效能和56倍Flops 是基於對手在用 E5-2697 v3 的

其他配套… 估計NV 在不久的將來會賣DGX 的雲

來點感想

1. 其實就是臺8x P100 的Blade, 28672 cuda cores 也就是11.2 片GTX1080

2. 效能會因為NVlink 變好, 在DL 方面, 這已經是暫時無法超越的伺服器節點

3. 拜Pascal 架構, DDR4和NVLink, 成績比起一年前竟提速了12倍, 我想這會成為NV 的伺服的的賣點

4. P100 的主要採購商Google 也是個設計主機很有名的主。這裡和Google 有很大的重疊。
